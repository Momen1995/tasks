#Project overview:
This is a portfolio website made with HTML and Tailwind CSS. It has a clean layout and works well on all screen sizes. The website has different sections like Home, About, Projects, and Contact. It is easy to use and shows a modern web design.

##Tech stack used: This project is developed using HTML and Tailwind CSS.

##Steps to run the project locally
1. clone the project-> https://github.com/Momen1995/tasks.git
2. Go to project folder -> cd project-name
3. Install dependencies -> npm install
4. Run the project locally -> npm run dev
